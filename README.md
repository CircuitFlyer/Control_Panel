# Control_Panel Remote Interface

Documentation and installation instructions.

The Control_Panel is a remotely mounted interface for CircuitFlyer active/governor inspired control line timer systems.  The hardware kit simplifies the installation of a pushbutton switch and addressable LED status indicator.
